阶段测测验一

判断题


1. linux 内核 1.0 的发布时间为 1994 年。(**对**)

2. Linux 支持几乎所有的网络协议，所以人们通常用它来作为服务器使用。(**对**)

3. 只有GPL协议是自由软件协议。(**错**)

4. Linux 兼容于 SYSV和BSD UNIX，并且符合 POSIX 标准。(**对**)

5. Linux 支持多种文件系统在文件级别上的互操作；不同种类的文件系统上的可执行文件是不能兼容执行的。(**错**)

6. 普通用户可以使用 logout 或 exit 关机；超级用户可以使用 shutdown 或 halt 关机。(**错**)

7. 不经用户同意，超级用户root 在操作系统中无权改变其他用户的登录口令。(**对**)

8. 命令 wc abc.txt 和 wc <abc.txt，无论输出结果还是运行机制均相同。(**错**)

9. 普通用户的Shell提示符是 #；超级用户的Shell提示符是 $ 。(**错**)

10. Linux 的内核是计算机硬件的首次扩充，它实现了操作系统的基本功能。(**对**)
